G A U S S

Gillespie Algorithm Used for Stochastic Simulation

Get Started

Easy

Upload constraints using a .react file.

Fast

Using a custom-made Gillespie Algorithm powered engine and our Django backend, GAUSS will get your results in a snap.

Analytical

GAUSS utilizes plotly.js, a powerful Javascript Library for graphing. Data can be downloaded as a .csv file.


A Powerful Engine. It'll blow your mind.

Our in-house, custom engine gets accurate results. Using the Gillespie algorithm and the Lotka-Volterra model, the engine can simulate an infinite number of reactions, each with different initial constraints.

Platform Agnostic. Run once, use anywhere.

GAUSS is a strong, independent web application that doesn't need many resources. The cloud-based platform will run your simulations, all from your favorite web-connnected device.

Beautiful.

GAUSS was carefully crafted with the latest Bootstrap. Our attention to detail ensures that your results come out looking their very best. Who says that computational biology can't be pretty?
